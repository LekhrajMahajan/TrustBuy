# TrustBuy — Frontend

This is the **React + Vite** frontend for the TrustBuy e-commerce platform.

For the full project documentation including technology stack, architecture, and file-by-file breakdown, see the **[root README](../README.md)**.

## Running Locally

```bash
npm install
npm run dev     # Starts on http://localhost:5173
```

## Build for Production

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run deploy
```
